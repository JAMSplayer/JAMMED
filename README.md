# JAMMED
An 8-bit platformer inspired by Jams Player, featuring retro-style gameplay and nostalgic pixel art.
CONTROLS: 
Move - A, D
Jump - Spacebar
Shoot - Q, Shift
Pause - E
Enjoy! https://jamsplayer.github.io/JAMMED/
